# Sanskrit Visual Builder v3.1 — Rust Edition

> Quantum-Classical Visual Programming IDE · Rust/WASM Engine

[![Tests](https://img.shields.io/badge/tests-71%2F71%20passing-brightgreen)](tests/)
[![Engine](https://img.shields.io/badge/engine-Rust%2FWASM-orange)](quantum_engine/)
[![License](https://img.shields.io/badge/license-MIT-blue)](LICENSE)
[![Blocks](https://img.shields.io/badge/blocks-528-cyan)](src/blocks/)

Sanskrit Visual Builder is a hybrid quantum-classical visual programming environment.
Drag-and-drop 528 blocks onto a canvas, connect them with wires, and run quantum
circuits, VQE chemistry calculations, Grover searches, and more — all powered by a
Rust/WebAssembly engine for 15-30× performance over pure JavaScript.

## Quick Start

### Prerequisites

| Tool | Version | Install |
|------|---------|---------|
| Node.js | ≥ 18 | https://nodejs.org (LTS button) |
| Rust | ≥ 1.65 | https://rustup.rs |
| Python | ≥ 3.8 | https://python.org (optional, for pre-computation) |

### Installation

```bash
# 1. Clone
git clone https://github.com/YOUR-ORG/sansqrit.git
cd sansqrit

# 2. Install Node.js dependencies
npm install

# 3. Install Rust WASM toolchain
rustup target add wasm32-unknown-unknown
cargo install wasm-pack

# 4. Build Rust engine → WASM (takes ~60s first time, ~5s after)
bash build.sh

# 5. Optional: Generate pre-computed lookup tables
python3 precompute.py

# 6. Run all tests — must show 71/71 passing
npm run test:all

# 7. Start
npm start
# → ✓ Sanskrit Rust/WASM engine loaded — ~15-30x speedup active
# → 📦 528 blocks loaded across 42 categories
# → Server running at http://localhost:3000
```

Open http://localhost:3000 in Chrome or Firefox.

## The Rust Engine

File: `quantum_engine/src/lib.rs` (442 lines)

### Architecture

```
QuantumRegister (QReg)
├── Shards (Vec<Shard>) — 10-qubit subsystems
│   └── Amps (sparse HashMap or dense Vec<f64>)
│       └── Complex64 (re: f64, im: f64)
├── resolve(q) → (shard_idx, local_qubit)
├── merge_shards() — triggered by cross-shard gates
└── All gates: H, X, Y, Z, S, T, Rx, Ry, Rz, CNOT, CZ,
               SWAP, iSWAP, RZZ, RXX, RYY, MS, Toffoli, QFT…

QAlg (algorithms)
├── grover(nQ, marked, shots) → HashMap<String, usize>
├── bell() → QReg
└── ghz(n) → QReg

WasmReg (#[wasm_bindgen]) — JavaScript interop wrapper
```

### All 15 Bug Fixes

| # | Gate/Function | Bug | Fix |
|---|---------------|-----|-----|
| 1 | `apply_1q` | Silent no-op on pure \|1⟩ states | `HashSet` of `idx & !stride` anchors |
| 2 | `CNOT` | Missed ctrl=1,tgt=1 pairs | Anchor from all ctrl=1 indices |
| 3 | `SWAP` | Only one orientation collected | Canonical anchor from both |
| 4 | `Toffoli` | Wrong for nQ > 3 | Set-based anchors |
| 5 | `merge_shards` | A bits placed in wrong position | `ia\|(ib<<na)` — A in LOW bits |
| 6 | `statevector` | Returned √prob instead of amplitude | Tensor-product actual {re,im} |
| 7 | `statevector` | MSB-first string convention | `string[i] = bit i of idx` (LSB-first) |
| 8 | `iSWAP` | SWAP+S+S gave wrong phase | H,CNOT(a→b),CNOT(b→a),H,S,S |
| 9 | `state_fidelity` | Missing complex conjugate on bra | `re += a1.re*b.re + a1.im*b.im` |
| 10 | `QFT` | Phase angle off by factor 2 | `2π/2^(k-j+1)` not `π/2^(k-j)` |
| 11 | Grover oracle | Only 2 controls for nQ > 3 | `n_controlled_z_all()` |
| 12 | Grover diffusion | `CZ(0,1)` only for nQ > 2 | Same n-controlled-Z fix |
| 13 | `RXX/RYY/MS` | Missing — caused NameError | Added via decomposition |
| 14 | `pauli_expectation` | Was null stub | Full implementation via clone+rotate |
| 15 | `QFT` arg order | Inverse flag received as nQ | Fixed function signature |

## Sanskrit DSL — Quick Reference

```sanskrit
-- Variables
let x = 42
mut counter = 0

-- Quantum
let q = qubits(4)
H(q[0])
CNOT(q[0], q[1])
Rx(q[2], PI/2)
let r = measure_all(q, shots=1000)
print(r.histogram)

-- Algorithms
let found = grover(4, [7], 2000)
molecule H2 { atoms:[H,H], bond_length:0.74, basis_set:"STO-3G" }
let e = vqe(H2, ansatz="UCCSD")

-- Math / Stats
print(gcd(48, 18))        -- 6
print(mean([1,2,3,4,5]))  -- 3.0
print(fibonacci(8))        -- [0,1,1,2,3,5,8,13,21]

-- F-strings
print(f"Energy: {e.energy:.6f} Ha")

-- List comprehensions
let squares = [x**2 for x in range(10)]
```

- **Rigetti Quil** — Native Rigetti QPU format
- **QIR** — LLVM-based cross-platform IR
- **SVG** — High-quality circuit diagram for papers
